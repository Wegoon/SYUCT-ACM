在家请客的二三事

出题人：江朋洁
知识点：其实没啥知识点，出题人说是小学数学题，好吧，那就是吧~~

Description

  听说题目越长，代码越短不知道是不是真的，总之，我们来讲一个故事吧。
  某一天，小www没有事干(总之就是非常的闲)(总之就是非常的闲)(总之就是非常的闲)，非常想请他的朋友到家里做客，不知道吃什么，所以选择制作黑暗料理。小王买了kkk瓶lllml的白酒(小孩子不能喝酒)(小孩子不能喝酒)(小孩子不能喝酒)，发现今天的砂糖大减价，买下了ppp克，来到水果店时，向年轻的老板娘要了ccc个新鲜的柠檬，老板娘告诉他每个柠檬切成ddd片会刚刚好，小www很开心，决定邀请nnn个朋友，并制作黑暗料理给朋友品尝。
  制作黑暗料理会用掉nlnlnlml的白酒，111片柠檬和npnpnp克的砂糖。
  众所周知，黑暗料理都不是给自己吃的。请问每个朋友最多能分到多少份黑暗料理？

Input Description

输入一行数据，包括888个正整数，分别为n,k,l,c,d,p,nl,np(1≤n,k,l,c,d,p,nl,np≤103)n,k,l,c,d,p,nl,np(1\leq n,k,l,c,d,p,nl,np\leq10^3 )n,k,l,c,d,p,nl,np(1≤n,k,l,c,d,p,nl,np≤103)，222个数间有111个空格。

Output Description

输出111个整数xxx，为每个朋友最多能分到的黑暗料理的份数。

Hint

  • 给朋友分黑暗料理当然不能偏心，哈哈哈~~

题解

题中说的是最多能做多少份，其实应该看最少的原料支持做多少份。只需要把最少的原料找到，再分给相应的人数。就得到答案了。

参考代码

#include <iostream>
using namespace std;
int main()
{
	int n, k, l, c, d, p, nl, np;
	cin >> n >> k >> l >> c >> d >> p >> nl >> np;
	cout << min(k * l / nl, min(c * d, p / np)) / n << endl;
	return 0;
}

你听说过史莱姆吗？？

出题人：江朋洁
知识点：二进制数的特点及简单应用

Description

  史莱姆(Slime)(Slime)(Slime)，是一种在现代电子游戏与奇幻小说常常出现的虚构生物，最早在195819581958年同名小说里出现。其流行形象是一种果冻状或半液体状、身体不透明或半透明、可以变换形状、能够分裂或融合的怪物。
  分裂史莱姆是一种可以分裂的史莱姆，每天夜晚，一个健康的史莱姆都可以分裂成两个健康的史莱姆。
  小Q小Q小Q重生成为了一个史莱姆，他想组建自己的史莱姆帝国，如果小Q小Q小Q想要拥有由n(1 ≤n≤ 109)n(1 ≤n≤ 10^9)n(1 ≤n≤ 109)个人组成的军队，还需要号召多少史莱姆加入他。

Input Description

输入一个数n(1 ≤n≤ 109)n(1 ≤n≤ 10^9)n(1 ≤n≤ 109)，nnn是小Q小Q小Q想要组成军队的人数。

Output Description

输出一个整数mmm，表示还需要mmm个史莱姆加入小Q小Q小Q，他才能实现这个目标。

Hint

看第二个样例，通过小Q小Q小Q分裂，1 −>2 −>4 −>81\ ->2\ ->4\ ->81 −>2 −>4 −>8，就可以得到888个史莱姆，所以不需要号召其他的史莱姆，所以答案是000。

题解

这是一道二进制题。举个例子：888的二进制码是100010001000，所以他能由一个史莱姆产生；555的二进制码是101101101，有两个“1”“1”“1”，因为一个史莱姆只能产生一个“1”“1”“1”（222的nnn次幂），所以需要两个史莱姆。找到规律后就可以写代码了！
第二种解法，把它当作一道数学题。如果一个数反复/222能够除到尽头（除到111），就说明它是222的nnn次幂，如果除到某个数得到的是奇数就加111，说明当前位置如果不再号召一个史莱姆就无法达成目标，然后也可以写代码啦（也不会超时）。

参考代码

二进制写法：

#include <iostream>
using namespace std;
int main()
{
    int n,ans=0;
    cin >> n;
    while (n)
    {
        if (n & 1)
            ans++;
        n >>= 1;
    }
    cout << ans-1 << endl;//减去它本身
    return 0;
}

数学算法：

            ans++;

    }

#include<iostream>
#include<cstdio>
#include<cmath>
#include<algorithm>
using namespace std;
int main()
{
    int n,a[100010],i,q,m;
    cin>>n;
    for(i=0; i<n; i++)
        cin>>a[i];
    sort(a,a+n);//用二分之前一定要先排序
    cin>>q;
    while(q--)
    {
        cin>>m;
        int l=0,r=n-1,mid;
        while(l<=r)//二分查找像小时候猜数一样自己可以尝试模拟一下
        {
            mid=(l+r)/2;
            if(m>=a[mid])
                l=mid+1;
            else
                r=mid-1;
        }
        cout<<l<<endl;
    }
    return 0;
}
//暴力时间复杂度是O(q*n)二分的时间复杂度是O(q*logn)

签到题

出题人：郭增静
知识点：其实没啥知识点，就是个简单的思维题吧，想明白就好~

Description

  动物园里的小骆驼问妈妈：“妈妈妈妈，为什么我们的睫毛那么长？”骆驼妈妈：“当风沙来的时候，长长的睫毛可以让我们在沙暴中看得到方向。”小骆驼又问：“妈妈妈妈，为什么我们的背那么驼，简直丑死了！”骆驼妈妈：“这个叫驼峰，它可以帮我们储存大量的水和养分，能让我们在沙漠里耐受十几天的无水无食条件。”小骆驼又问：“妈妈妈妈，为什么我们的脚掌那么厚？”骆驼妈妈：“那可以让我们重重的身子不至于陷在软软的沙子里，这样便于我们长途跋涉啊！”小骆驼高兴坏了：“哇，原来我们这么有用啊！可是妈妈，为什么现在我们还在动物园里，而不去沙漠远足呢？”
  好吧，只是让你们看个故事。
  现在小骆驼给出一个数字n(1≤n≤1018)n(1\leq n\leq10^{18})n(1≤n≤1018)，他想知道能否找到两个整数a,ba,ba,b(1≤a,b≤n)(1\leq a,b\leq n)(1≤a,b≤n)，使得 gcd(a,b)=1\ gcd(a,b)=1 gcd(a,b)=1，并输出a,ba,ba,b的最大和。

Input Description

输入一个整数nnn。

Output Description

输出一个整数mmm，为a,ba,ba,b的最大和。

Hint

  • 好吧，偷偷告诉你aaa可以等于bbb。
  • 好吧，不用怀疑，肯定存在a,ba,ba,b。

题解

本题其实是有个坑点的，就是n=1的情况，但是出题人贵人多忘事，忘记加那个数据了。也怪我，我验题的时候问了出题人一嘴，然后就没有特意去用错误代码去验题，上面我给你们的提示aaa可以等于bbb其实就是暗示你们n=1的时候，a=b=1a=b=1a=b=1，害白瞎了一个题( •̥́ ˍ •̀ू )

#include<cstdio>
#include<cmath>
#include<iostream>
#include<algorithm>
#include<cstring>
#include<set>
#include<vector>
using namespace std;
typedef long long ll;
int i, j, k;
int l, r, mid, n, d;
int a[1111111];
int left(int x)
{
    l = 0, r = n - 1;
    while (l <= r)
    {
        mid = (l + r) / 2;
        if (a[mid] < x)
            l = mid + 1;
        else
            r = mid - 1;
    }
    return l;
}
int right(int x)
{
    l = 0, r = n - 1;
    while (l <= r)
    {
        mid = (l + r) / 2;
        if (a[mid] > x)
            r = mid - 1;
        else
            l = mid + 1;
    }
    return r;
}
int main()
{
    cin >> n >> d;
    for (i = 0; i < n; i++)
        cin >> a[i];
    sort(a, a + n);
    ll sum = 0;
    for (i = 0; i < n; i++)
    {
        int L = left(a[i] - d);
        int R = right(a[i] + d);
        sum += R - L;
    }
    cout << sum / 2 << endl;
    return 0;
}

#include <iostream>

long long n,a,dp[1000005];
int main()
{
    ios::sync_with_stdio(0),cin.tie(0),cout.tie(0);
    cin>>n;
    dp[0]=0;
    for(int i=1; i<=n; i++)
    {
        cin>>a;
        if(i==1)
            dp[i]=max(dp[i-1],a);
        else
            dp[i]=max(dp[i-1],dp[i-2]+a);
    }

}

本题其实核心就在于短除法，我们都知道10进制转2进制就是不停对商除以2然后取余直到商为0，然后倒序输出余数。没错，这就是短除法。这里只不过我们把它们推广到了任意进制上，然后模拟这个不停除、取余的过程。。。

参考代码

#include <iostream>
#include <algorithm>
using namespace std;
int x,y,len1,len2,a[200005],b[200005];
string s1,s2;
void char_int();//char->int
void int_char();//int->char
void change();//核心代码，短除法实现进制转化
int main()
{
    ios::sync_with_stdio(0),cin.tie(0),cout.tie(0);
    cin>>x>>s1>>y;
    len1=s1.size();
    char_int();
    change();
    int_char();
    cout<<s2<<endl;
    return 0;
}
void char_int()//char->int
{
    for(int i=0; i<len1; i++)
    {
        if(s1[i]<='9')
            a[i]=s1[i]-'0';
        else if(s1[i]>='A'&&s1[i]<='Z')
            a[i]=s1[i]-'A'+10;
        else
            a[i]=s1[i]-'a'+36;
    }
}
void int_char()//int->char
{
    for(int i=len2-1; i>=0; i--)
    {
        if(b[i]<10)
            s2+=b[i]+'0';
        else if(b[i]<36)
            s2+=b[i]-10+'A';
        else
            s2+=b[i]-36+'a';
    }
}
void change()//核心代码，短除法实现进制转化
{
    int st=0;
    len2=0;
    while(st<=len1)
    {
        int temp=0;
        for(int i=st; i<len1; i++)
        {
            a[i]+=temp*x;
            temp=a[i]%y;
            a[i]/=y;
        }
        b[len2++]=temp;
        while(st<=len1&&a[st]==0)
            st++;
    }
}
